Fullstack Auth App

Tech Stack

- Frontend: Next.js
- Backend: Node.js + Express
- Database: MongoDB
- State Management: Zustand

Features

- User Registration
- User Login (JWT)
- Protected Routes
- Auth State Management

API Endpoints

- POST /api/auth/register
- POST /api/auth/login
- GET /api/users/profile (Protected)
